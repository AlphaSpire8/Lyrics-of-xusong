# 许嵩正式专辑歌词整理

本仓库用于按正式个人专辑整理许嵩歌曲歌词，并为后续歌词内涵分析预留目录。

## 收录范围

- 仅收录许嵩正式个人录音室专辑中的歌曲。
- 不收录未归入正式专辑的单曲、EP、精选集、影视/游戏推广曲、现场版本或合作特别版本。
- 歌词文本应来自授权来源，或由仓库维护者在确认版权许可后补充。

## 目录说明

- `lyrics/`：按专辑年份与名称划分的歌词文件。
- `analysis/`：歌词内涵分析预留目录，当前阶段暂不开展具体分析。
